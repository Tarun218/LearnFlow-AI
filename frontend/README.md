# LearnFlow AI — Frontend

Next.js App Router client for [LearnFlow AI](../README.md).

## Quick start

```bash
npm install
cp .env.example .env.local
# Edit NEXT_PUBLIC_API_URL
npm run dev
```

See the [root README](../README.md) for full setup, deployment (Vercel), and environment variables.

## Deploy on Vercel

- **Root directory:** `frontend`
- **Env:** `NEXT_PUBLIC_API_URL` = your backend URL (no trailing slash)
